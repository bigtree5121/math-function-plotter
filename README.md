# 初中数学函数绘图工具

一个专为初中学生设计的数学函数绘图网页工具，支持多种常见函数类型的可视化展示。

## 🎯 功能特点

### 支持的函数类型
- **一次函数**: `y = ax + b` (如: `2x + 1`)
- **二次函数**: `y = ax² + bx + c` (如: `x^2 - 4x + 3`) 
- **反比例函数**: `y = k/x` (如: `3/x`)
- **绝对值函数**: `y = |x|` (如: `abs(x-2)`)
- **平方根函数**: `y = √x` (如: `sqrt(x)`)
- **复合函数**: 支持基本运算组合

### 交互功能
- 实时函数绘制
- 多函数同时显示（最多5个）
- 坐标系缩放和平移
- 网格线显示/隐藏
- 清除和重置功能
- 函数颜色自动区分

## 📝 使用说明

### 基本操作
1. 在输入框中输入函数表达式
2. 点击"添加函数"按钮
3. 函数图像会自动在坐标系中显示

### 函数输入格式
```javascript
// 一次函数
2*x + 1      // y = 2x + 1
x - 3        // y = x - 3

// 二次函数  
x^2          // y = x²
x^2 + 2*x + 1   // y = x² + 2x + 1
-0.5*x^2 + 4    // y = -0.5x² + 4

// 反比例函数
1/x          // y = 1/x
-2/x         // y = -2/x
3/(x-1)      // y = 3/(x-1)

// 其他函数
abs(x)       // y = |x|
sqrt(x)      // y = √x
(x-2)^2      // y = (x-2)²
```

### 操作技巧
- **缩放**: 使用鼠标滚轮或触摸手势
- **平移**: 按住鼠标左键拖拽
- **重置**: 点击"重置视图"恢复默认视图
- **清除**: 点击"清除全部"删除所有函数

### 快捷键
- `Enter`: 快速添加函数
- `Escape`: 清除当前输入
- `Ctrl/Cmd + Z`: 删除最后添加的函数

## 🐛 故障排除

### 常见问题
**Q: 函数不显示图像？**
A: 检查函数语法，确保使用正确的运算符 (`*` 表示乘法, `^` 表示乘方)

**Q: 图像显示不完整？**
A: 尝试调整坐标系范围或重置视图

**Q: 在手机上操作不便？** 
A: 使用双指缩放和单指拖拽进行操作

**Q: 函数计算错误？**
A: 确保输入的是数学上有意义的函数（如√x中x≥0）

### 错误提示
- `语法错误`: 检查括号匹配和运算符使用
- `数学错误`: 检查函数定义域（如除零、开方等）
- `超出范围`: 调整坐标系显示范围

## 📚 扩展开发

### 添加新函数类型
在 `math-parser.js` 中的 `functions` 对象添加新函数：

```javascript
functions: {
    // 现有函数...
    'sin': Math.sin,    // 正弦函数
    'cos': Math.cos,    // 余弦函数
    'log': Math.log,    // 对数函数
}
```

### 自定义样式
修改 `styles.css` 中的CSS变量：

```css
:root {
    --primary-color: #your-color;
    --grid-color: #your-grid-color;
    --function-colors: #color1, #color2, ...;
}
```
