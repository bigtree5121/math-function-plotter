# 初中数学函数绘制工具产品需求文档（PRD）

## 1. 项目概述

### 1.1 项目名称
初中数学函数图像绘制工具

### 1.2 项目背景
为了帮助初中学生更直观地理解数学函数的图形特征和性质，开发一个在线的函数图像绘制工具，让学生能够输入函数方程并立即看到对应的图像，提升数学学习效果。

### 1.3 目标用户
- 主要用户：初中学生（13-16岁）
- 次要用户：初中数学教师、家长辅导者
- 使用场景：课堂教学、课后练习、家庭辅导

### 1.4 核心价值
- 将抽象的函数概念可视化
- 提供即时反馈的学习体验
- 支持多种函数类型的标准绘制
- 操作简单，学习成本低

## 2. 功能需求分析

### 2.1 支持的函数类型
| 函数类型 | 标准形式 | 示例 | 优先级 |
|---------|---------|------|--------|
| 一次函数 | y = ax + b | y = 2x + 1 | P0 |
| 二次函数 | y = ax² + bx + c | y = x² - 2x + 1 | P0 |
| 反比例函数 | y = k/x | y = 6/x | P0 |
| 正比例函数 | y = kx | y = 3x | P0 |
| 常数函数 | y = c | y = 5 | P1 |
| 绝对值函数 | y = a|x + b| + c | y = |x - 1| + 2 | P1 |
| 平方根函数 | y = a√(x + b) + c | y = √x | P2 |

### 2.2 核心功能模块

#### 2.2.1 函数输入模块 (P0)
**功能描述：**
- 提供文本输入框供用户输入函数表达式
- 支持标准数学表达式格式
- 实时语法检验和错误提示
- 提供常用函数模板快速选择

**详细需求：**
- 输入格式支持：y = 表达式格式
- 支持运算符：+, -, *, /, ^, (), |, √
- 支持函数：abs(), sqrt()
- 变量支持：x, y
- 实时预览输入的数学公式
- 语法错误时给出明确提示信息

#### 2.2.2 坐标系绘制模块 (P0)
**功能描述：**
- 绘制标准的直角坐标系
- 支持坐标轴标签和刻度
- 网格线显示开关
- 坐标系缩放和平移

**详细需求：**
- 默认显示范围：x轴 [-10, 10]，y轴 [-10, 10]
- 主要刻度间隔：1个单位
- 次要网格线：0.5个单位间隔
- 坐标轴标签：显示数值
- 原点标识：清晰标记(0,0)点
- 支持鼠标滚轮缩放
- 支持鼠标拖拽平移

#### 2.2.3 函数曲线绘制模块 (P0)
**功能描述：**
- 根据函数方程绘制精确曲线
- 支持多条曲线同时显示
- 不同颜色区分不同函数
- 处理函数定义域限制

**详细需求：**
- 曲线绘制精度：每像素采样
- 支持同时绘制最多5条函数曲线
- 预设颜色方案：蓝、红、绿、橙、紫
- 自动处理不连续点（如反比例函数）
- 超出显示范围的部分正确截断

#### 2.2.4 交互功能模块 (P1)
**功能描述：**
- 鼠标悬停显示坐标值
- 点击查看精确坐标
- 特殊点标注（如顶点、截距）
- 函数表达式显示开关

**详细需求：**
- 鼠标悬停实时显示(x,y)坐标
- 点击曲线显示精确数值
- 自动标注关键点（顶点、与轴交点）
- 函数列表显示，支持隐藏/显示
- 清除单条或全部函数曲线

### 2.3 高级功能需求 (P2)

#### 2.3.1 参数调节功能
- 滑块调节函数参数（如y = ax + b中的a、b）
- 实时预览参数变化对图形的影响

#### 2.3.2 导出功能
- 导出图像为PNG格式
- 保存函数列表为文本文件

#### 2.3.3 预设模板
- 提供常见函数的快速模板
- 经典例题库

## 3. 技术方案设计

### 3.1 技术架构
**前端技术栈：**
- HTML5 + CSS3 + JavaScript (ES6+)
- Canvas API 或 SVG 用于图形绘制
- 数学解析库：Math.js 或自研解析器
- 响应式设计框架：Bootstrap 或 Tailwind CSS

**部署方案：**
- 纯前端静态网页
- 支持本地离线使用
- CDN加速访问

### 3.2 核心算法设计

#### 3.2.1 数学表达式解析
```javascript
// 表达式解析流程
1. 词法分析：将输入字符串分解为token
2. 语法分析：构建抽象语法树(AST)
3. 语义分析：检查数学合法性
4. 表达式求值：给定x值计算y值
```

#### 3.2.2 函数绘制算法
```javascript
// 曲线绘制策略
1. 自适应采样：根据函数变化率调整采样密度
2. 不连续点检测：识别垂直渐近线等
3. 平滑连线：使用线性插值或贝塞尔曲线
4. 边界裁剪：处理显示范围外的部分
```

### 3.3 性能要求
- 函数解析响应时间 < 100ms
- 图形绘制完成时间 < 200ms
- 支持1000个以上采样点的流畅绘制
- 内存使用 < 50MB

```
│   - 模板     │                           │

├─────────────┴───────────────────────────┤
│                 状态栏                    │
└─────────────────────────────────────────┘
```

### 4.2 具体界面设计

#### 4.2.1 函数输入区域
- 输入框占位符："请输入函数，如：y = x^2 + 2*x + 1"
- 实时预览当前输入的数学公式（MathJax渲染）
- 添加函数按钮，最多支持5个函数
- 语法错误提示用红色标注

#### 4.2.2 快速模板区域
```
常用模板：
[一次函数] [二次函数] [反比例] [绝对值]
y=x+1     y=x²      y=1/x    y=|x|
```

#### 4.2.3 绘图控制区域
- 坐标轴范围调节：X轴 [-□ to □] Y轴 [-□ to □]
- 网格开关：☑ 显示网格
- 重置视图按钮
- 清除所有函数按钮

#### 4.2.4 函数列表区域
```
当前函数：
1. y = x² + 1    [蓝色] [👁] [🗑]
2. y = 2x - 3    [红色] [👁] [🗑]
   └─ 显示/隐藏   删除
```

### 4.3 响应式设计
- 桌面端：侧边栏 + 主绘图区
- 平板端：上下布局，控制面板折叠
- 手机端：单页面，通过标签切换功能

## 5. 用户体验设计

### 5.1 操作流程设计
1. **新用户引导**
   - 首次访问显示操作提示
   - 提供示例函数进行演示

2. **输入体验优化**
   - 智能提示常用函数格式
   - 支持键盘快捷键（Enter提交）
   - 历史输入记录

3. **错误处理**
   - 友好的错误提示信息
   - 提供修正建议
   - 不阻断用户操作流程

### 5.2 辅助功能
- 支持键盘导航
- 高对比度模式
- 字体大小调节
- 色盲友好的配色方案
